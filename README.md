My Online Resume

> [!TIP]
> How to publish?
> 1. Access repository "Settings" tab
> 2. Access "Pages" tab
> 3. Select branch (master) for deployment and click "Save"

> [!TIP]
> How to republish?
> 1. Access repository "Action" tab
> 2. Select the workflow
> 3. Click on "Re-run all jobs" button.
